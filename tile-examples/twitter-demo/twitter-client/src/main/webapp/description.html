Twitter Heatmap and Sentiment Analysis Demo

This Aperture Tiles demo illustrates the integration of advanced analytics into an interactive tile-based visual analytics demo. The demo features multiple tiles layers:

  1. A base google map
  2. A heatmap of geo-coded twitter messages ("tweets") across North America
  3. Interactive tile carousels summarizing the top key words used in tweets and machine inferred positive / neutral / negative sentiment across each tile region

The dataset used for this demonstration consists of 1,220,635 Twitter messages occurring on January 2nd, 2014.
