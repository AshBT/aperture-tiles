Twitter Latin American Topic Summarization Demo

This Aperture Tiles demo illustrates the integration of advanced analytics into an interactive tile-based visual analytics demo. The demo features multiple tiles layers:

  1. A base google map
  2. A heatmap of geo-coded twitter messages ("tweets") across South America
  3. Interactive tile carousels summarizing topics in tweets and language across each tile region

The dataset used for this demonstration consists of 66,996,907 Twitter messages occurring on between February and April, 2014.
